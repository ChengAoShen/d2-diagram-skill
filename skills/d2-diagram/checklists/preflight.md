# Preflight Checklist

- D2 installed (`d2 version` works).
- Input scope is clear: system, boundaries, and desired output format.
- Layout engine decision made (`dagre` default, `elk`/`tala` as needed).
- Theme decision made (light only or light+dark).
- Required artifacts identified (`svg`/`png`/`pdf`/`pptx`/`gif`/`txt`).
- File paths chosen and writable.
