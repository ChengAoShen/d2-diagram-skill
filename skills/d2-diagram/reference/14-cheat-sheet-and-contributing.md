# Cheat Sheet and Contributing

## Fast Remember List

- connection ops: `--`, `->`, `<-`, `<->`
- shape defaults to rectangle
- use `vars` + `${}` for reuse
- run `d2 validate` then `d2 fmt`
- use `d2 -w` for iterative authoring

## Contributing Guidance

- keep examples runnable
- keep docs concise and command-driven
- add references when introducing non-obvious behavior
