# API, CLI, Studio

## CLI Highlights

- compile/render: `d2 in.d2 out.svg`
- validate: `d2 validate file.d2`
- format: `d2 fmt file.d2`
- open playground: `d2 play file.d2`
- list layouts/themes: `d2 layout`, `d2 themes`

## D2 Studio

Use Studio for collaborative editing and visual iteration.

## API

Use D2 API mode when integrating rendering into automated pipelines.
