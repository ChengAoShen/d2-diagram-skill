# Reference Index

How to use this index while executing tasks:

- New diagram: start with `03-core-basics.md`, then `05-customization.md`.
- Dense graph readability issues: go to `06-layouts.md` and `07-in-depth.md`.
- Multi-board storytelling: use `08-composition.md`.
- Reusable organization across files: use `09-imports-patterns.md`.
- Export/runtime problems: use `12-exports.md` and `13-faq-troubleshooting.md`.

- `01-introduction.md`: purpose, scope, philosophy.
- `02-install-run-watch.md`: install, run, validate, format, watch.
- `03-core-basics.md`: shapes, connections, containers.
- `04-special-objects.md`: text/code/icons/sql/classes/sequence/grid.
- `05-customization.md`: themes, styles, classes, dimensions, positions, animation.
- `06-layouts.md`: dagre, elk, tala tradeoffs.
- `07-in-depth.md`: strings, vars, globs, comments, overrides, models.
- `08-composition.md`: layers, scenarios, steps, linking, composition formats.
- `09-imports-patterns.md`: imports and advanced organization patterns.
- `10-extensions-ecosystem.md`: official integrations and community ecosystem.
- `11-api-cli-studio.md`: API, CLI highlights, Studio entry points.
- `12-exports.md`: all export modes and caveats.
- `13-faq-troubleshooting.md`: common failures and fixes.
- `14-cheat-sheet-and-contributing.md`: quick references and contribution links.
- `15-icon-library.md`: icons and image usage playbook.
