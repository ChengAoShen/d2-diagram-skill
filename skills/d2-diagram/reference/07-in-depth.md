# In Depth

Topics included:

- strings and quoting rules
- comments
- `vars` and `${}` substitutions
- globs (`*`, `**`, `***`) and filters
- overrides
- models and legend
- auto-formatter behavior

## High-impact Practices

- Use `vars` for repeated values.
- Use globs to set defaults globally.
- Use `d2 fmt` after large edits.
- Prefer clarity over clever syntax.
